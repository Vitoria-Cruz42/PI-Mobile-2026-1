# HemoConecta

## Projeto Integrador 2026/1

O **HemoConecta** é um aplicativo desenvolvido como Projeto Integrador 2026/1 com o objetivo de integrar informações sobre doações de sangue e medula óssea, facilitando o processo e tornando-o mais acessível para todos.

## Conteúdo do Repositório

Este repositório contém todos os arquivos relacionados ao desenvolvimento e à apresentação do projeto, organizados nas seguintes categorias:

* **Prototipação**

  * Prints das telas desenvolvidas no Figma.
  * Link para o protótipo interativo.

* **Modelagem**

  * Modelo Entidade-Relacionamento (MER).

* **Documentação**

  * PDF da apresentação do trabalho.
